//// TASK MANAGER CLI ////

Dia 1 => Task Manager CLI: python task_manager/main.py - Criação de um menu interativo que adiciona tarefas do usuário,lista as tarefas e  determina o estado das tarefas.

Dia 2 => Persistência em JSON - Agora o programa salva as tarefas e seus estados na maquina usando json  diretorio(tasks.json).

Dia 3 => Listagem em tabela Rich e melhorias de UX - Agora a tabela segue estilos das recomendações de uso de markup do Rich. E visuavelmente falando, a tabela ficou um pouco mais "clean" com a coluna "Status" e seus valores.

Dia 4 => Editar e remover tarefas - Adicionado mais 2 opções na tabela, Editar = consegue alterar o nome da tarefa, e, Remover = consegue remover a tarefa por completo da tabela. 

Dia 5 => Busca e filtro por status - Adicionada 2 novas funcionalidades na tabela onde é possível buscar uma tarefa por palavra chave/específica e outra função de filtro por status - Concluídas ou Pendentes.
